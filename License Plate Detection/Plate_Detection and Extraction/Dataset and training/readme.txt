The Dataset used to train the License Plate detection model is taken from Kaggle 
"https://www.kaggle.com/datasets/saisirishan/indian-vehicle-dataset"
This dataset contained images of License Plates of multiple states all across India.
The dataset was annotated manually using MakeSense.ai and trained using YOLOv8
